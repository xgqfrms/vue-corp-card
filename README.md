# vue-corp-card

> An Vue UI Card Component used for display Crop infos!

```sh
# install cli
$ yarn global add @vue/cli
$ yarn global add @vue/cli-service-global

# uninstall cli
$ yarn global remove @vue/cli

```

```sh
# dev build tools
$ yarn add -D bili rollup-plugin-vue vue-template-compiler

```
